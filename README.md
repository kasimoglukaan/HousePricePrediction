# House Price Prediction (Machine Learning)

This project is a machine learning application designed to estimate house prices based on property features such as size, building age, number of rooms, and location characteristics.

The model analyzes real estate data and predicts housing prices using several machine learning algorithms. The goal of the project is to explore how machine learning can be applied to real estate price prediction.

---

# Features

- House price prediction based on property characteristics
- Data cleaning and preprocessing
- Feature engineering
- Multiple machine learning models
- Hyperparameter tuning with GridSearchCV
- Model performance comparison
- Visualization of prediction results
- Feature importance analysis
- Streamlit interface for interaction

---

- Python
- Pandas

- XGBoost
- Matplotlib
- Seaborn

---

# Machine Learning Models

The following models were implemented and compared:

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

Model performance was evaluated using:

- Mean Squared Error (MSE)
- R² Score

---

# Data Processing

The dataset contains housing information including:

- Gross square meters
- Net square meters
- Building age
- Number of rooms
- Number of bathrooms
- Number of balconies
- Floor information
- Heating type
- Credit eligibility

The data was cleaned and transformed before training the models. Outliers were removed and numerical scaling was applied.
